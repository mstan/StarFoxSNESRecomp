# Star Fox SNES Recomp

An LLE-first recompilation project for **Star Fox (USA), revision 2** on the
Super NES. The 65816 architectural model and the Super FX/GSU coprocessor are
authoritative; ahead-of-time compilation is an optional optimization.

The project expects a user-supplied, unheadered 1 MiB ROM with these hashes:

- CRC32: `8FC4E6D0`
- MD5: `DEF66DB12F5E644C0CF00C42CFA7AE7B`
- SHA-256: `82E39DFBB3E4FE5C28044E80878392070C618B298DD5A267E5EA53C8F72CC548`

ROM data is not part of this repository.
